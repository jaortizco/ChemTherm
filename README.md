# ChemTherm

ChemTherm is a Python package related to chemical thermodynamics.

At the moment, only chemical equilibrium of a gas mixture is supported.

## Supported especies:
Carbon moxide <br>
Carbon dioxide <br>
Chlorodifluoromethane <br>
Hydrogen <br>
Methane <br>
Nitrogen <br>
Oxygen <br>
Steam <br>
